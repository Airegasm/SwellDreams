# chip-tool Installation

Place `chip-tool.exe` in this directory.

## Download

Get the latest Windows build from:
https://github.com/project-chip/connectedhomeip/releases

Look for:
- `chip-tool-windows-x64.zip` or
- `connectedhomeip-windows-*.zip`

Extract `chip-tool.exe` to this folder.

## Verify

The file structure should be:
```
backend/bin/chip-tool/chip-tool.exe
```

SwellDreams will verify this on startup.
